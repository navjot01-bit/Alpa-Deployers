# Alpa-Deployers

CMPT 2500 ML project using NPRI data.

## Folders
- src - code files
- data/raw - raw data
- data/processed - cleaned data and predictions
- models - saved model
- config - config files
- tests - test files
- notebooks - notebook work

## Files
- preprocess.py
- train.py
- predict.py
- evaluate.py

## Run
- make preprocess
- make train
- make predict
- make evaluate
- make test
